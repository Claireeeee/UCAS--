## README

- model文件夹：用open nmt生成的翻译模型

- train data文件夹：生成过程用到的训练数据

  - src_ans.txt：答案集合
  - src_ques.txt：问句集合
  - valid_ans.txt：用于验证答案集合
  - valid_ques.txt：用于验证的问句集合

- test文件夹：模型的正确率测试生层的文件

  一共测试3次，分别保存在test1/2/3文件夹下：

  - test_ques.txt文件：用来测试的50个问句
  - test_ans.txt文件：50个问题的标准答案
  - pred.txt：模型翻译的结果

- lab report：实验报告
